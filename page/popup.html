팝업 호출

.ui-pop-open 클래스로 열기
* a태그를 사용하여 href에 대상 팝업의 id 지정
<a href="#popupLogin" class="button ui-pop-open">버튼</a>


* popOpen(대상 팝업의 id) 혹은 popOpen(대상 팝업의 id, 팝업 닫을때 포커스 줄 대상)
<button class="button" onclick="popOpen('#popupLogin',this)">버튼</button>

팝업 닫기

* 대상 팝업의 id 내에 .ui-pop-close클래스의 버튼으로 닫기
<button class="button ui-pop-open">버튼</button>

* popClose(대상 팝업의 id)
<button class="button" onclick="popClose('#popupLogin')">버튼</button>
